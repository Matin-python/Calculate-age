📌 Age Calculator (Python)

📖 Project Overview

This project is a simple Age Calculator written in Python.
It calculates a person's exact age in:

Years

Months

Days

Total Months

Total Days

Total Hours

Total Seconds


The project includes two versions:

1. Basic Version – Manual validation and custom calculations


2. Improved Version – Cleaner implementation using Python’s datetime module



The purpose of this project is to demonstrate:

Input validation

Date handling

Logical problem-solving

Code refactoring and improvement

🚀 How to Run

Make sure you have Python 3 installed.

Run the program:

python basic_version.py

or

python improved_version.py
